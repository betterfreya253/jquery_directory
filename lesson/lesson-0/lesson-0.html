jQuery Introduction and Installation

What is jQuery?

Answer

JavaScript library for web development that is fast, lightweight and feature-rich It simplifies HTML document traversal, event handling, animation, and AJAX interaction.

What tasks can you perform with jQuery?

Answer

With jQuery, you can select and manipulate elements, handle events, create animations, make AJAX requests, and add interactivity to websites.

What is the advantage of jQuery's cross-browser compatibility?

Answer

jQuery abstracts browser differences, ensuring consistent code behavior across various platforms and eliminating the need for writing browser-specific code.

What is the philosophy behind jQuery?

Answer

jQuery follows a "write less, do more" philosophy, providing easy-to-use methods and utilities that streamline web development tasks.

How to use jQuery locally?

Answer

If you want to use jQuery locally, download it and reference it in head tag

<head>
 <script src="jquery-1.11.3.js"></script>
</head>

How to use jQuery without download?

Answer

Do NOT want to download jQuery? Put the jQuery link in script tag instead.

<head>
 <script src="http://code.jquery.com/jquery-latest.js"></script>
</head>
